# Notes
## Data types of char values
##### Characters
values that represent one single character
|C++ Keyword|Size|range of values|
|-|-|-|
|char|1 byte|0 to 127|
- Note: uses ASCII

##### Character Variables
use `char` to represent data that only requires a single character
-`chat input;`
- `char marital_status`
